# Projeto 1: Site de Empresa

Projeto desenvolvido com `HTML` e `CSS`, com foco em layout institucional, visual moderno e responsividade para diferentes tamanhos de tela.

## Sobre o projeto

Este site foi criado como modelo de pagina institucional para empresa, apresentando:

- secao inicial com destaque principal
- area "sobre" com missao, visao e valores
- secao de servicos
- indicadores da empresa
- area de contato

## Tecnologias utilizadas

- HTML5
- CSS3

## Estrutura do projeto

```bash
.
|-- index.html
|-- style.css
|-- README.md
```

## Como executar

1. Baixe ou clone este repositorio.
2. Abra o arquivo `index.html` no navegador.

## Responsividade

O projeto foi desenvolvido com tecnicas de responsividade para funcionar bem em:

- desktops
- tablets
- celulares

## Objetivo

Servir como atividade pratica de desenvolvimento web, aplicando conceitos de estrutura HTML, estilizacao com CSS e adaptacao para dispositivos moveis.
